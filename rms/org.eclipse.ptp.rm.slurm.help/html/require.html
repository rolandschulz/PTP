SLURM Resource Manager for PTP

Requirements

In addition to having Eclipse and PTP installed, you need to have the following software packages installed.

  • CDT 7.0, available from the Eclipse download site
  • SLURM 2.1 statble release, or SLURM 2.2 (under development, at the time of this writing), available from LLNL SLURM website

You should review the following information about SLURM installation and usage

  • SLURM User Guide
  • SLURM Install Guide
