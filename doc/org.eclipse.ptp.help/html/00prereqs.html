PTP - Prerequisites

The PTP Release 1.0 requires the base Eclipse platform, the CDT or FDT, and MPI.

  • Java 1.4.2 or later
  • Eclipse platform: Version 3.1.1
  • C or Fortran programming environment. One of the following:
    • CDT C/C++ Development Toolkit for Eclipse Version 3.0.2
    • FDT (for Fortran, or also works for C/C++ files. Wherever CDT is mentioned, FDT projects and files, for C/C++, should also work).
    • Photran Eclipse plug-ins for Fortran
  • MPI - Support for MPI is currently only available using Open MPI 1.0.2 or later.
