PTP - Prerequisites

The following prerequisites are required for PTP 4.0 installation. For more detailed information, including installation instructions, see http://wiki.eclipse.org/PTP/release_notes/4.0

  • Java 1.5 or later
  • Eclipse SDK version 3.6 or later
  • Eclipse C/C++ Development Tools (CDT) version 7.0.0 or later
  • Remote Development Tools 3.2 or later

PTP supports a number of parallel runtime systems and job schedulers. These need to be installed on the target system prior to using PTP:

  • Open MPI version 1.2 or later (preferred version is 1.4.x)
  • MPICH2 version 1.0.6p1 and later
  • IBM Parallel Environment (PE)
  • IBM LoadLeveler (LL)
  • SLURM
  • PBS
