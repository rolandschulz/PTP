PTP - Prerequisites

The PTP Release 1.1 (November 2006) requires the base Eclipse platform, the CDT or FDT, and MPI.

  • Java 1.5 or later
  • Eclipse platform: Version 3.2 or later
  • C or Fortran programming environment. One of the following:
    • CDT C/C++ Development Toolkit for Eclipse Version 3.1 or later
    • FDT (for Fortran, or also works for C/C++ files. Wherever CDT is mentioned, FDT projects and files, for C/C++, should also work).
    • Photran Eclipse plug-ins for Fortran
  • MPI - Support for MPI is currently only available using Open MPI version 1.0.2 or 1.2
