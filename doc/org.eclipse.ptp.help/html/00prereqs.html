PTP - Prerequisites

The PTP Release 2.0 (early 2007) requires the base Eclipse platform, the CDT, and MPI.

  • Java 1.5 or later
  • Eclipse platform: Version 3.3.1.1 or later
  • C or Fortran programming environment. One of the following:
    • CDT C/C++ Development Toolkit for Eclipse Version 4.0.2 or later
    • Photran (for Fortran, or also works for C/C++ files. Wherever CDT is mentioned, Photran projects and files, for C/C++, should also work, although this is not very thoroughly tested).
    • Photran Eclipse plug-ins for Fortran
  • MPI - Support for MPI is currently only available using Open MPI version 1.0.2 or 1.2
