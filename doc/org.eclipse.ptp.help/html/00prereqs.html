PTP - Prerequisites

The PTP requires the base Eclipse platform, the CDT or FDT, and MPI.

  • Eclipse platform: Version 3.1
  • C or Fortran programming environment. One of the following:
    • CDT C/C++ Development Toolkit for Eclipse Version 3.0.0
    • FDT (for Fortran, or also works for C/C++ files. Wherever CDT is mentioned, FDT projects and files, for C/C++, should also work).
    • Photran Eclipse plug-ins for Fortran

     

  • MPI - Support for MPI is currently only available using Open MPI (see below).

    Parallel Runtime

    • Support for running parallel applications is currently only available using the Open Runtime Environment component of Open MPI.
    • Please download the latest version of Open MPI from http:/www.open-mpi.org and follow the installation instructions.
    • Install in any convenient location. Use the Open RTE preference page to specify the installation location of the Open RTE daemon (usually $ompi_install/bin/orted).
