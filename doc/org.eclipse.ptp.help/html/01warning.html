Warning dialogs on first project

The first time you bring up the PTP Runtime perspective, you may see two warning dialogs appear.

The first popup warning indicates that the number of machines is not set. You can set this in Preferences next.

Press OK.

You may also see a popup warning that the control or monitoring system is not detected.

Press OK. You can also set this in Preferences.

If you need to change the default values for these settings, use the PTP Preferences.
