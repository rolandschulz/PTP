PTP Developers Guide

This book is aimed at developers wishing to understand more about the internal details of how the Parallel Tools Platform works. It provides the following information:
  • An overview of the PTP architecture, including the internal model, system services, and the proxy system.
  • A description of the APIs for interacting with model elements, launching jobs, debugging, etc.
  • A description of the extension points provided by PTP to allow developers to add extra functionality.
  • Some tutorials showing some examples of using this information.
