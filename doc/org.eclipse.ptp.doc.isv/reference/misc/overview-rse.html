RSE Overview - Map of RSE Plug-ins

The Remote System Explorer provides a comprehensive environment for managing remote resources and processes

The Remote System Explorer tooling itself is divided up into a number of separate plug-ins. The following table shows which API packages are found in which plug-ins as of Eclipse 3.0. This table is useful for determining which plug-ins a given plug-in should include as prerequisites.

API Package Required plug-in id

org.eclipse.rse.ui[.*] org.eclipse.rse.ui
