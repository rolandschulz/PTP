Remote C/C++ New Project Wizard

Configurations




Name Function
Project type Selected project type.
Tool-chains One or more selected tool chains
Configurations List of all build configurations to be created. Check items you want to create. At least one configuration must be checked.
Select all Checks all configurations
Deselect all Unchecks all configurations
Advanced settings ... Displays properties dialog for new project. It is recommend that you do no click this button as it may take several minutes for the properties dialog to appear. These properties can also be edited after the project has been created.
