PTP - Prerequisites

The following prerequisites are required for PTP 3.0 installation. For more detailed information, including installation instructions, see http://wiki.eclipse.org/PTP/release_notes/3.0

  • Java 1.5 or later
  • Eclipse SDK version 3.5 or later
  • Eclipse C/C++ Development Tools (CDT) version 6.0.1 or later (Remote Development Tools requires CDT 6.0.2 or later)

PTP supports a number of parallel runtime systems and job schedulers. These need to be installed on the target system prior to using PTP:

  • Open MPI version 1.2.x and 1.3.x
  • MPICH2 version 1.0.6p1 and later
  • IBM Parallel Environment (PE)
  • IBM LoadLeveler (LL)
  • SLURM
  • PBS (monitoring only)
