Project Name

For Project name, type in "SPU". You can select the appopriate Project Type (Cell PPU Executable, Cell SPU Static Library, etc...) for the project you are creating. When you select a project type, you will see the default Configurations available for that project type. Later in the tutorial you will learn how to create new build configurations, as well as how to modify the existing built-in configurations' settings.

For Project Type, select Cell SPU Executable and click Finish.
