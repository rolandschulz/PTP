Create Local Cell Simulator

Now that the projects are created and properly configured, we can test our program, but first, a Cell environment must be created and started. The Cell IDE integrates IBM's full-system simulator for the Cell BE processor into eclipse, so that your only a few clicks away from testing your application on a Cell environment, even if you don't have access to a physical Cell processor.

In the Remote Environments view at the bottom, right click on Local Cell Simulator and select Create.
