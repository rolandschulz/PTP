Select Debugger and Launch Debug Configuration

In the Debugger field, you can choose from Cell PPU gdbserver, Cell SPU gdbserver , or Cell BE gdbserver. To debug only PPU or SPU programs, select Cell PPU gdbserver or Cell SPU gdbserver, respectively. The Cell BE gdbserver option is the combined debugger, which allows for debugging of PPU and SPU source code in one debug session.

Set the Debugger field to Cell BE gdbserver then click Debug to launch the debug session.
