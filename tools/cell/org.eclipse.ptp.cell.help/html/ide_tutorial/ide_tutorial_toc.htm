PTP Cell/B.E IDE - Tutorial

This introductory walk-through explores IBM's SDK for Multicore Acceleration IDE and offers a click-for-click lesson on how to use most of the main features of the IDE.

In this tutorial

  • Step-by-step instructions on how to create, build, and run PPU/SPU managed make projects
  • How to use the Local Cell Simulator environment
  • Walk-through of how to use the performance analysis tools (static and dynamic)
  • How to configure and use the C/C++ Cell Target Application launcher to run/debug your Cell applications
  • Example usage scenario of the IDE's Accelerated Library Framework IDE wizard
  • And more...
