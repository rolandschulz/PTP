Introduction to IBM SDK for Multicore Acceleration Integrated Development Environment

IBM SDK for Multicore Acceleration Integrated Development Environment (IDE), which is built upon the Eclipse and C Development Tools (CDT) platform, integrates the Cell BE GNU tool chain, compilers, IBM Full-System Simulator for the Cell BE, and other development components in order to provide a comprehensive, user-friendly development platform that simplifies Cell BE development. Key features include the following:
  • a C/C++ editor that supports syntax highlighting; a customizable template; and an outline window view for procedures, variables, declarations, and functions that appear in source code
  • a rich visual interface for PPE (Power Processing Element) and SPE (Synergistic Processing Element) GDB (GNU debugger)
  • seamless integration of simulator into Eclipse
  • automatic builder, performance tools, and several other enhancements
