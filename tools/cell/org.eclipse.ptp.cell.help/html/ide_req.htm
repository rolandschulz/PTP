Platform Requirements

Operating System: Linux®
Architecture: x86, x86_64, PPC, PPC_64, Cell BE
Software:

  • Fedora 8
  • Cell BE SDK 3.0
  • Eclipse 3.2 (not compatible with Eclipse 3.3)
  • C/C++ Development Tools (CDT) 4.0
  • Java™ 1.5 (IBM's or Sun's.)
    • Note: The Java™ that comes installed in Fedora (GCJ) is not sufficient.
