SimulatorControl Parameters

Default values:

  • req: The value is required. No default value is possible. The parameters must be assigned.

  • ---: Has no default value, since the parameters is calculated by the object that implements ISimulatorParameters.

  • tclSD: Tcl Script Default. This parameter is optional. If assigned to null, then the TCL script will use its own default value. Typically, if this parameter is passed as environment variable, the the variable will not be set.

Uses:

  • Tcl (env): Passes as environment variable to the TCL script. This implies use by SimulatorControl to create the list of environment variables to create the simulator process.

  • Ctrl: Used by SimulatorControl for internal logic. This parameter is not passed to the simulator.

  • Ctrl (deploy): Used by SimulatorControl during the deploy of files. This parameter is not passed to the simulator.

  • Ctrl (launch): Used by SimulatorControl to create the simulator process. This parameter is not passed to the simulator.

  • Ctrl (cleanup): Used by SimulatorControl during the cleanup. This parameter is not passed to the simulator.

  • Ctrl (cmdline): Used by SimulatorControl build the commandline.
