shmem_addr_accessible

Purpose

Determines whether an address is accessible via IBM openshmem data transfers operations from the specified processing element (PE).

C syntax

#include <shmem.h>
 
int shmem_addr_accessible(void *addr, int pe);

Parameters

INPUT
addr
The local address to be queried.
pe
The processing element number of the PE that will access the local address.

Description

The shmem_addr_accessible function returns 1 if the specified addr is a symmetric data object; otherwise it returns 0.

Symmetric data objects, also known as remotely accessible objects, are arrays or variables that exist with the same size, type, and effective address on all PEs. The following data objects are remotely accessible:

The variables allocated by shmalloc, shmemalign, or shrealloc functions.

Non-stack C and C++ variables, if the same executable file is used for all PEs
