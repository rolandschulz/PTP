Setup for PTP OpenMP Tools Plug-in

Note: in order to have the OpenMP tools recognize #pragmas, you must install a slight fix to CDT.

CDT 3.0.x Bugzilla entry # 133723 contains the 2-line patch. Without this, the rest of the plug-in functions, but pragmas will not be recognized. This patch will be included in the next (3.1) release of CDT.

The basic setup for the OpenMP tools is the same as for the MPI tools setup. Essentially, point CDT at the header files and at the command to call your compiler with the OpenMP information.
