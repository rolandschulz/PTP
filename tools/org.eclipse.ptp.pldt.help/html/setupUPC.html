Setup for PTP UPC Tools

The basic setup for the UPC tools is the same as for the MPI and OpenMP tools setup. Essentially, point CDT at the header files and at the command to call your compiler with the UPC information.

The UPC preferences should be set to point to your UPC header file.

You may need to manually alter the project build properties (in the project properties) to add compiler and linker flags specific to UPC to your project.

 

Now set up to run the UPC features.
