Service Model

The service model is a representation of:

  • What Services are defined for the IDE to take advantage of
  • What Service Providers are available for those services
  • How those services are configured for each project


    • The service model can be configured in several ways:

  • During project creation in the New Remote C/C++ Project wizard
  • In the Remote Services project property page
  • In the Convert To Remote Project wizard


    • In order for RDT remote projects to function properly, a service model must be configured for your project.
