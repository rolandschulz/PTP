Remote C/C++ Perspective

A perspective is a layout of views (development tools) in the Workbench window. Each type of perspective is a combination of views, menus, and toolbars that enable you to perform a particular task.

Selecting / Opening Views:
  • You can add views to a perspective. From the menu bar choose Window > Show View > Other and select a new view from the Show View dialog.
  • To reset the current perspective to its original layout, from the menu bar choose Window > Reset Perspective.

The Remote C/C++ development tools contribute the following perspective to the workbench:

Remote C/C++ perspective views

This perspective is tuned for working with remote C/C++ projects. By default it consists of an editor area and the following views:

  • Project Explorer
  • Remote Systems
  • Outline
  • Console
  • Problems
  • Remote Call Hierarchy
  • Remote Type Hierarchy
  • Remote Shell
