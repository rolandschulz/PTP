Service Providers




Name Function
Service Services are essentially well defined sets of functionality which can be provided by Service Providers.
Service Provider Service Providers provide implementations of a Service. Each service has a list of available providers. The NULL service provider is selected by default and does nothing. To select a service provider click on the table cell and a list of providers will appear. Each service provider must be configured separately.
Provider Configuration Shows details of the service provider configuration, such as the name of the connection that the provider has been configured to use. If the provider has not yet been configured then Not configured will be shown.
Configure Each service provider must be configured separately. Select a service provider then click the Configure button, a dialog will appear that will allow the provider to be configured.
